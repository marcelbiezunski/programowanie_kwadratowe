# Metody Optymalizacji: Programowanie Kwadratowe

Repozytorium zawiera materiały na laboratorium z przedmiotu Metody Optymalizacji. 
Tematem zajęć jest Programowanie Kwadratowe (QP) oraz wykorzystanie metody zbiorów aktywnych (Active Set Method) w środowisku MATLAB.

## Struktura Repozytorium

Repozytorium podzielone jest na dwa główne katalogi:

    zadania – Zawiera szablony kodu dla studentów. Są to pliki .m z lukami do uzupełnienia (np. definicje macierzy H, A, b).

    odpowiedzi – Zawiera w pełni działające, rozwiązane skrypty.

Wymagania

    MATLAB (R2018b lub nowszy zalecany)

    Optimization Toolbox (funkcja quadprog)
